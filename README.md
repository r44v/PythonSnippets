# Tools

I became a big fan of [pipenv](https://pipenv.readthedocs.io/en/latest/) but lately I started using the venv module more.

```
python -m venv venv

# Ubuntu
venv/bin/activate
# Windows
venv\Scripts\activate

pip install -r requirements.txt
```
